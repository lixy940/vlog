## 使用说明
1. 下载代码
`git clone https://github.com/gujiniCY/mayday.git`
2. 运行sql文件夹下的sql文件
3. 到src/main/resources下的application.yaml下修改你的数据库链接
4. 保存好以后 mvn package打包
5. 进入target文件夹运行nohup java -jar vlog.jar &

### 如果使用又拍云cdn的话请到参数跟随里面设置全程跟随以及开启重定向跟随
